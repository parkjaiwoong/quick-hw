# 급송·야간 할증 — 시장 비교

## 1. 일반 퀵서비스 시장

| 구분 | 급송/즉시 | 기본/일반 | 야간 (20:00~08:00) |
|------|-----------|-----------|--------------------|
| **예상시간** | 30분 내외 | 1~3시간 | - |
| **요금** | 기본요금 + 10~30% 할증 | 기본 요금 | 기본요금 + 20~50% 할증 |
| **비고** | 시장에서 급송은 당연히 할증 | 일반 요금 | 오후 8시~다음날 오전 8시 |

## 2. 적용 전 QuickHW (기존)

| 구분 | 급송 | 기본 | 야간 |
|------|------|------|------|
| **예상시간** | 30분 | 3시간 | - |
| **요금** | **기본과 동일** | 기본요금 + 거리 등 | **미적용** |
| **비고** | 할증 없음 | - | 야간 구분 없음 |

## 3. 적용 후 QuickHW (현재)

| 구분 | 급송 | 기본 | 야간 |
|------|------|------|------|
| **예상시간** | 30분 | 3시간 | - |
| **요금** | 기본+거리+물품 후 **+20% 할증** | 기본요금 + 거리 + 물품 | 기본+거리+물품 후 **+20% 할증** |
| **야간 구간** | - | - | **20:00~08:00 (KST)** |
| **적용 시점** | 고객이 급송 선택 시 | - | 픽업 시각 기준 (즉시=현재, 예약=선택 일시) |

### 산식
- 기본요금 + 거리요금 + 물품 가산 + 무게 가산 + 크기 가산  
- 급송이면 위 합계에 20% 추가  
- 야간이면 위 합계에 20% 추가 (급송·야간 중첩 시 둘 다 적용)

## 4. 시장 대비 요약

| 항목 | 시장 | QuickHW (적용 후) |
|------|------|-------------------|
| 급송 할증 | 10~30% | 20% (중간값) |
| 야간 할증 | 20~50% | 20% (보수적) |
| 야간 구간 | 20:00~08:00 | 20:00~08:00 동일 |
| 야간 적용 기준 | 픽업 시각 | 픽업 시각 동일 |
